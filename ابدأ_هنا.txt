═══════════════════════════════════════════════════════════════
🚀 طريقتان للرفع:
═══════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│ الطريقة 1: تلقائي (موصى بها للمبتدئين)                    │
└─────────────────────────────────────────────────────────────┘

1. انقر نقراً مزدوجاً على ملف: RUN_THESE_COMMANDS.bat
2. اتبع التعليمات على الشاشة
3. أدخل اسمك وبريدك الإلكتروني
4. أدخل اسم المستخدم على GitHub
5. استخدم Personal Access Token عند طلب كلمة المرور

┌─────────────────────────────────────────────────────────────┐
│ الطريقة 2: يدوي (للمحترفين)                               │
└─────────────────────────────────────────────────────────────┘

افتح Terminal/Command Prompt في مجلد المشروع ونفذ:

1. إعداد Git (مرة واحدة فقط):
   git config --global user.name "اسمك"
   git config --global user.email "your.email@example.com"

2. إضافة الملفات:
   git add .

3. عمل Commit:
   git commit -m "Initial commit: Complete Musify app"

4. ربط بـ GitHub (استبدل YOUR_USERNAME):
   git remote add origin https://github.com/YOUR_USERNAME/musify.git

5. رفع المشروع:
   git branch -M main
   git push -u origin main

═══════════════════════════════════════════════════════════════
🔑 مهم جداً: Personal Access Token
═══════════════════════════════════════════════════════════════

عند رفع المشروع، لا تستخدم كلمة المرور العادية!
استخدم Personal Access Token:

1. اذهب إلى: https://github.com/settings/tokens
2. اضغط "Generate new token" → "Generate new token (classic)"
3. اختر "repo" في الصلاحيات
4. انسخ الـ Token
5. الصقه عند طلب Password

⚠️ احفظ الـ Token في مكان آمن!

═══════════════════════════════════════════════════════════════
🆘 إذا واجهت مشاكل:
═══════════════════════════════════════════════════════════════

المشكلة: "remote origin already exists"
الحل:
   git remote remove origin
   git remote add origin https://github.com/YOUR_USERNAME/musify.git

المشكلة: "Authentication failed"
الحل:
   - استخدم Personal Access Token وليس كلمة المرور
   - تأكد من أن الـ Token لديه صلاحيات "repo"

المشكلة: "failed to push"
الحل:
   git pull origin main --allow-unrelated-histories
   git push -u origin main
